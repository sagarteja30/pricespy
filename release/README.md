# PriceSpy AI - Chrome Extension

## Installation

1. Download and unzip `pricespy-extension.zip`
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the unzipped folder
5. Visit any Amazon product page to see price predictions

## Features

- Real-time price prediction on Amazon product pages
- Buy/Hold/Sell recommendations
- 7-day and 30-day price forecasts
- Dark mode support
- Multiple API fallbacks for reliability

## Requirements

- Chrome browser (version 88+)
- Internet connection for API calls

## Troubleshooting

If predictions don't appear:
- Refresh the Amazon page
- Check that the extension is enabled
- Try disabling other price-related extensions
