**🚀 I built an AI that predicts Amazon prices 14 days ahead!**

Ever wondered if you're buying at the right time? I did too. So I built PriceSpy AI 🔮

**What it does:**
✅ Installs on your browser in seconds
✅ Automatically detects Amazon product pages
✅ Analyzes price history using ML
✅ Tells you: BUY NOW or WAIT

**The tech stack:**
⚡ FastAPI + PostgreSQL (backend)
🤖 Prophet + NumPy (price prediction)
📦 Chrome Extension Manifest V3 (frontend)
☁️ Railway (deployment)
🔷 Supabase (database)

**How it works:**
1. Daily scraper collects prices from Amazon bestsellers
2. Stores historical prices in PostgreSQL
3. Uses linear regression + Prophet ML
4. Predicts if price will rise/fall in 14 days
5. Shows confidence score based on data quality

**Results:**
📊 1 day data = 50% confidence
📈 5+ days data = 92% confidence

**Why I built this:**
Most price trackers show you WHAT happened. I wanted to know WHAT WILL HAPPEN. So I built it. 💡

🔗 Live Demo: https://pricespy-production-c1dd.up.railway.app
📦 GitHub: https://github.com/sagarteja30/pricespy

Would love your feedback! 👇
